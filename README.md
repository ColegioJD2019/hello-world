# Bienvenido al repositorio del Colegio  "Juan Diego"

Bienvenido al espacio de almacenamiento del proyecto web del colegio privado de educación básica y diversificado "Juan Diego", del municipio de Comitancillo, del departamento de San Marcos.

## Repositorio

En este espacio es donde se mantiene almacenado todo lo relacionado a la información básica y general del proyecto basado en la creación, diseño e implementación del sistema de control del Colegio "Juan Diego", dicha institución actualmente trabaja con los niveles Básico y Diversificado, por lo cual este sistema llevará al registro del control de cada una de las actividades que con estos niveles educativos se refiere.
